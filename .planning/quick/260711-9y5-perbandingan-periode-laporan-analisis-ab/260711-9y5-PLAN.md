# Quick Task 260711-9y5: Perbandingan Periode + ABC + Ekspor CSV + Sweep Istilah

## Objective

Melengkapi dua ciri MIS terakhir yang direkomendasikan (pelaporan komparatif antar-periode dan analisis
Pareto/ABC), menambah ekspor CSV data terolah, dan — permintaan eksplisit user — menghapus singkatan
yang tidak umum dari seluruh antarmuka: "KPI" → "Indikator Kinerja", "ETA" → "Estimasi Tiba", "SLA" →
"Tenggat". Nama variabel/endpoint internal tidak diubah (tidak terlihat pengguna).

## Tasks

### Task 1: Sweep istilah singkatan di semua string yang tampil ke pengguna

**Files:** `src/pages/Dashboard.jsx`, `src/pages/Laporan.jsx`, `src/pages/StatusDistribusi.jsx`,
`src/pages/Landing.jsx`, `src/utils/mis.js`, `server/src/services/misService.js`,
`server/src/services/targetKpiService.js`, `server/src/services/ringkasanService.js`

**Action:** Ganti "KPI"/"ETA"/"SLA" pada label kolom, judul modal/section, pesan toast, teks kartu,
pesan notifikasi & judul tindakan backend, dan entri activity log. Komentar kode dan identifier tetap.

**Verify:** `grep` string UI tidak lagi memuat KPI/ETA/SLA; build sukses.

### Task 2: Bagian "Perbandingan Periode" di Laporan

**Files:** `src/utils/mis.js` (helper murni `computePerbandinganPeriode`), `src/pages/Laporan.jsx`

**Action:** Kartu berisi tabel/baris metrik periode terpilih vs periode sebelumnya (total permintaan,
volume keputusan dibuat, pengiriman selesai, ketepatan estimasi tiba) dengan delta % dan panah
naik/turun; mengikuti pilihan periode halaman (minggu-ini/bulan-ini; "semua" menampilkan keterangan
tidak berlaku). Ikut dirender di laporan cetak bila murah.

**Verify:** Angka helper diuji cepat via node terhadap data seed; build sukses.

### Task 3: Analisis ABC (Pareto) di AnalisisRanking

**Files:** `src/pages/AnalisisRanking.jsx`

**Action:** Klasifikasi kota kelas A (kumulatif ≤ 80% permintaan), B (≤ 95%), C (sisa); chip kelas pada
tabel ranking + kartu ringkasan "Analisis Pareto (Klasifikasi ABC)" berisi jumlah kota & kontribusi per
kelas dan pesan fokus manajemen.

**Verify:** Build sukses; kelas konsisten dengan kumulatif.

### Task 4: Ekspor CSV data terolah

**Files:** `src/pages/AnalisisRanking.jsx`, `src/pages/Dashboard.jsx`

**Action:** Tombol "Ekspor CSV" (downloadCsv dari utils/csv.js) pada: tabel Peramalan Permintaan,
tabel Skor & Alokasi (AnalisisRanking), dan riwayat harian di kartu Tren Kinerja (Dashboard Manajer).

**Verify:** Build sukses (downloadCsv sudah teruji dipakai halaman lain).
